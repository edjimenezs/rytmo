API Flow Diagram:
-----------------
1. Connect Flow:
   User clicks "Connect Strava"
   → GET /api/strava/auth
   → Redirect to Strava OAuth
   → User authorizes
   → Strava redirects to /api/strava/callback
   → Exchange code for tokens
   → Save to database
   → Auto-sync last 30 days
   → Redirect to dashboard

2. Sync Flow:
   User clicks "Sync Now"
   → POST /api/strava/sync
   → Get valid access token (auto-refresh if needed)
   → Fetch activities from Strava API
   → Save/update in database
   → Return success + count

3. Disconnect Flow:
   User clicks "Disconnect"
   → Confirm dialog
   → POST /api/strava/disconnect
   → Revoke token at Strava
   → Delete from database
   → Return success

Strava API Endpoints Used:
---------------------------
Authorization:
  GET  https://www.strava.com/oauth/authorize
  POST https://www.strava.com/oauth/token

Activities:
  GET  https://www.strava.com/api/v3/athlete/activities
  GET  https://www.strava.com/api/v3/activities/:id

Deauthorization:
  POST https://www.strava.com/api/v3/oauth/deauthorize
